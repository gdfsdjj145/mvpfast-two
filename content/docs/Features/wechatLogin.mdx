目前支持 PC 和 H5 的微信登录方式，使用微信授权登录。

##### 配置

```json
NEXT_PUBLIC_WECHAT_OPEN_APPID=""
WECHAT_OPEN_APPID=""
WECHAT_OPEN_APPSECRET=""
```

移动端的登录还到公众号进行以下配置。
