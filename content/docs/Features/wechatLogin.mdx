目前支持 PC 和 H5 的微信登录方式，PC 使用扫码的方式登录，H5 使用网页授权的方式。

##### 配置

```json
NEXT_PUBLIC_WECHAT_APPID=xxxx // 微信公众号的appid
```

移动端的登录还到公众号进行以下配置。

![image-20241120151052471](/docs/assets/wechatLogin3.png)

#### 扫码登录

扫码登录流程：扫码->公众号回复->登录。

![image-20241120150601354](/docs/assets/wechatLogin1.png)

#### H5 登录

H5 登录流程：点击微信登录->跳转至微信浏览器->授权->登录。

![image-20241120150742049](/docs/assets/wechatLogin2.png)
